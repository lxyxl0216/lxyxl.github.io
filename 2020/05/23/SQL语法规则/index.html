  • SQL指令需要语句结束符，默认;
  • 如果使用到关键字或者保留字，需要用反引号

  • 数据库名字不可修改（先新增，后迁移，最后删除）

SQL表(字段)操作

创建数据表

  • 表与字段不分家
  • 创建需要制定存储的数据库
  • 字段至少需要制定名字和类型
  • 数据库不限定字段数量
    • 每个字段使用逗号分隔
    • 最后一个不需要字段
  • 表可以制定表选项
    • 存储引擎：engine[=] 具体存储引擎
    • 字符集：[defalut] charset 具体字符集（继承数据库）
    • 校对集：collate (继承数据库)

存储引擎是指数据存储和管理的方式

  • Innodb：默认存储引擎，支持事务处理，数据统一管理
  • Mylsam：不支持事务处理。数据、表结构、所以独立管理。MySQL5.6以后不再维护。

如果想创建一个人与已有表一样的数据表create table 表名 like 数据库名字.表名

显示数据表

  • 数据表的显示与用户权限有关
  • 两种方式
    • 显示所有表
    • 显示具体数据表的创建指令

查看数据表

  • 通常是查看字段信息
  • 详细的显示字段的各项信息
  • 查看语法有三种

更改数据表

  • 修改表名：rename
  • 修改表选项：alter table 表名

更改表名：rename table 原名 to 新名

更改字段

  • 字段操作包括：字段名字，类型，属性。
  • 字段操作包括4类
    • 新增字段：add [column]
    • 更改字段名：change
    • 修改类型：modify
    • 删除字段：drop

删除数据表

  • 字段删除会将数据也删除

SQL数据操作

新增数据

  • 新增数据是根据表的字段顺序和数据类型要求将数据存放到数据表中
  • 数据表中的数据以行为存储单位，实际存储属于字段存储数据
  • 数据插入分两种方式
    • 全字段插入：insert into 表名 value(字段列表顺序对应的所有值)
    • 部分字段插入：insert into表名(字段列表) values(字段列表对应的值顺序列表)

查看数据

  • 将表中存在的数据按照指定的要求显示
  • 数据显示包含字段名和数据本身
  • 数据查看分成两种方式
    • 查看全部字段：使用*代替所有字段
    • 查看部分字段：明确字段名
  • 查看数据很多也是根据条件查询部分

更新数据

  • 更新数据通常是根据条件更新某些数据，而不是全部记录都更新

删除数据

  • 删除通常是有条件的删除
  • 删除为不可逆

数据源

where子句

  • 跟在from数据源之后，对数据进行条件匹配
  • where在磁盘读取后，进入内存之前进行筛选

运算符

比较运算符

  • >,<,=,>=,<=,<>
  • between A and B;
  • in（数据1，数据2.。。。）
  • like ‘pattern’：像pattern
  • _对应位置的任意一个字符
  • %匹配当前位置往后任意数量的字符串

逻辑运算符

  • and ，or，not

null运算符

  • is null，is not null

group by

  • 分组统计，根据某个字段将所有结果进行分类，并进行数据统计分析
  • group by 字句一定是出现在where后
  • 统计函数：
    • group_concat():将组里的某个字段全部保留
    • any_value():不属于分组字段的任意一个组里的值
    • count():求对应分组的记录数量
    • sum():
    • max()/min()
    • avg()

回溯统计

在进行分组时，每一次结果的回溯都进行一次汇总统计。

with rollup

分组排序

  • group by 字段名 [ASC]; 默认；
  • group by 字段名 DESC;

having

  • 条件筛选数据

  • having 字句必须出现在group by字句之后

  • 针对内存里已经加载的数据
  • having 能做where能做的所有事情，反正不行。

order by

  • 按照某一个条件进行排序
  • 参照物是校对集
  • order by [ASC]
  • order by DESC

limit

  • 限制数据获取的数量
  • limit 必须在order by之后
  • limit数量：限制获取的数量
  • limit 起始位置，数量：限制数据获取的位置以及数量；
